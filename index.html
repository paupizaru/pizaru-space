Work Experience

Software Tester / QA Analyst

IBM - Chile | May 2021 - Apr 2022

My most recent experience!. It was a banking project. Our team focused on regression testing of the bank's website, specifically on the client view.
The main duties were:

  • To develop and document test plans
  • To design and to execute test cases
  • To analyze and document test results
  • To identify, to report and to track bugs found on the system
  • To register the SQA process in pre productive environment using ALM Octane.

Different tools and software products were used such as:

  • Jira
  • Soap-UI
  • Postman
  • PostgreSQL
  • Sybase
  • MS SQL Server

A little bit of test automation was carried out using Gherkin (Cucumber), and automated end-to-end tests were executed through Selenium (Java - IntelliJ).

Software Tester / QA Analyst

Moofwd - Chile | Mar 2021 - Apr 2021

The company has many mobile app projects with universities around the globe, so the main task was to do mobile testing to those apps, specially exploratory and smoke testing.
Related duties were:

  • To execute, analyze and document results of software application tests
  • To develop and to document test plans
  • To identify, to report, and to track defects and bugs through Jira software
